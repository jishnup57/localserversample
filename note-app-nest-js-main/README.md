## Description

Download & install [Node JS](https://nodejs.org/en/)

Install yarn using flollowing command

```bash
$ npm install --global yarn
```

add sudo first in any permission error shown (Linux & Mac)

Open project inside visual studio code
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start:dev

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
